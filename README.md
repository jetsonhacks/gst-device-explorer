# gst-device-explorer

`gst-device-explorer` is planned as a Jetson-aware GStreamer device and
pipeline exploration tool for video inputs, audio inputs, and audio outputs on
Jetson and generic Linux systems.

The project is currently in the design and specification phase. No application
code, CLI, GUI, probing implementation, or pipeline execution support is
complete yet.

See the `docs/` directory for the initial specification, architecture notes,
development principles, and Milestone 1 scope.
